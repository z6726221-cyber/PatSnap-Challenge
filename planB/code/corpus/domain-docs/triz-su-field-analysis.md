# TRIZ 物场分析（Su-Field Analysis）

**物场分析**（Substance-Field Analysis，简称 Su-Field）是 TRIZ 中用来建模"问题结构"的图形化工具，把任何技术系统的最小功能单元抽象为三个元素：

- **S1（作用物质/对象）**：被作用的对象。
- **S2（工具物质）**：施加作用的工具或载体。
- **F（场）**：连接 S1 和 S2 的作用方式，可以是机械场、热场、电场、磁场、化学场等。

## 三种基本模型

1. **完整有效模型**：S2 通过 F 对 S1 施加了期望的作用，系统正常工作。
2. **有害作用模型**：S2 通过 F 对 S1 产生了不期望的副作用，需要消除或屏蔽。
3. **不足/缺失模型**：S1 需要某种作用但 F 或 S2 缺失/不足，需要补全。

## 用法

物场分析常与 **76个标准解**（Standard Solutions）配套使用：把问题画成物场模型后，对照标准解模板（如"引入 S3 消除有害作用""改变场类型增强作用"等）快速定位改造方向。这是与"40个发明原理"平行的另一条 TRIZ 工具线——40原理面向"技术矛盾"，物场分析+标准解面向"功能结构缺陷"。

## 边界

- 物场分析假设问题能被抽象为"物质-场"三元结构，对纯信息类、组织流程类问题的适用性较弱，通常需要先转化为物理/工程视角。
