# 智慧芽 Engineering Agents 产品能力

智慧芽官网将其 AI Agent 产品线归入 **Patsnap Eureka** 平台，按用途分为几类，其中与研发问题求解直接相关的是：

## Engineering Agents（工程智能体）

面向 R&D 问题求解场景，官网明确列出的能力方向包括：

- **Functional Analysis（功能分析）**：对产品/系统的功能结构建模，定位问题所在的功能环节。
- **Root-cause / TRIZ-based Ideation（根因分析与基于TRIZ的创新构想）**：结合根因分析方法与 TRIZ 方法论，辅助工程师产出创新解决方向。

## 平台定位

智慧芽把整套 Agent 能力（IP Search Agents / IP Drafting Agents / Engineering Agents / Life Sciences Agents / Materials Agents 等）映射到研发生命周期的四个阶段：**Concept（构思）→ Discovery（发现）→ Protection（保护）→ Commercialization（商业化）**，对应 Ask / Research / Solve / Draft / Validate 等动作。

## 与 TRIZ 语料的关系

本语料库收录的 TRIZ 方法论文档（矛盾矩阵、40发明原理、物场分析、IFR、专利挖掘结合应用）是 Engineering Agents 场景中"root-cause/TRIZ-based ideation"能力可能涉及的方法论基础，用于本次 Demo 演示"技术问答"能做到什么程度的知识底座——具体产品功能实现细节以官网/产品文档为准，本语料不代表功能承诺清单。

## 边界

- 本文档仅摘录官网公开介绍文字，未接触内部产品路线图或未发布功能。
- "83x more accurate than ChatGPT"等营销性宣称官网有提及但缺乏可验证的第三方评测方法说明，本语料不采信此类未经核实的营销数字，仅记录产品方向定性描述。
