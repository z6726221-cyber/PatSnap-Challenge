# LexisNexis PatentSight+

**PatentSight+** 是 LexisNexis 旗下的 IP 分析平台，面向专利组合估值、竞争情报和战略决策，隶属于 LexisNexis Intellectual Property Solutions，目标客户包括企业、律所、顾问、投资者、政府和学术机构。[来源: patentsight-plus, 抓取于 2026-07-08]

## 主要能力

- **数据资产**：经整理校正的专利所有权数据（"pre-corrected ownership data"），配实时法律状态追踪。
- **Patent Asset Index**：官网称为"全球技术实力和影响力的客观衡量指标"的专有指数。
- **组合分析**：组合基准对比、技术分类、竞争格局分析。
- **场景覆盖**：竞争情报、组合优化、并购尽调、许可/SEP分析、风险规避、技术侦察。
- **可持续发展关联分析**：把专利映射到联合国可持续发展目标（SDG）。
- **协作工具**：告警、多语言界面、PPT导出分享。

[来源: patentsight-plus, 抓取于 2026-07-08]

## AI 能力：Protégé

官网核心 AI 功能是 **Protégé**，宣称是"专为专利分析打造"的助手，支持用自然语言提问而非手动构建查询。官网描述其能"结构化分析、整合业务背景，交付透明、可决策的洞察"。提及的具体应用场景包括：厘清所有权结构（挖出"控制实体、隐藏关联和潜在风险"），把组合问题直接转化为带可追溯推理步骤的结构化分析。[来源: patentsight-plus, 抓取于 2026-07-08]
