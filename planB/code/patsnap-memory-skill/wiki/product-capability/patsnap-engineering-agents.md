# 智慧芽 Engineering Agents

智慧芽官网将其 AI Agent 产品线归入 **Patsnap Eureka** 平台，按用途分类，其中与研发问题求解直接相关的是 **Engineering Agents（工程智能体）**。[来源: patsnap-engineering-agents, 2026-06-20, 核实于 2026-07-08]

## 官网列出的能力方向

- **Functional Analysis（功能分析）**：对产品/系统的功能结构建模，定位问题所在的功能环节。
- **Root-cause / TRIZ-based Ideation（根因分析与基于TRIZ的创新构想）**：结合根因分析方法与 TRIZ 方法论（详见 [[triz-overview]]），辅助工程师产出创新解决方向。

[来源: patsnap-engineering-agents, 2026-06-20]

## 平台定位

智慧芽把整套 Agent 能力（IP Search Agents / IP Drafting Agents / Engineering Agents / Life Sciences Agents / Materials Agents 等）映射到研发生命周期四个阶段：**Concept（构思）→ Discovery（发现）→ Protection（保护）→ Commercialization（商业化）**，对应 Ask / Research / Solve / Draft / Validate 等动作。[来源: patsnap-engineering-agents, 2026-06-20]

## 待核实

- 官网提及"83x more accurate than ChatGPT"等营销性数字宣称，缺乏可验证的第三方评测方法说明，本知识库**不采信**此类未经核实的营销数字，仅记录产品方向的定性描述。
- 具体"root-cause/TRIZ-based ideation"在产品内的实现细节（是否覆盖矛盾矩阵/物场分析等具体工具）未在官网首页找到更细颗粒度说明，需要更细致的产品文档核实，暂标"待核实"。
