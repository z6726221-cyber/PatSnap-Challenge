# TRIZ 40个发明原理

TRIZ 从数万份专利中归纳出的 **40个发明原理**，用于在识别出技术矛盾后给出通用的解决方向。[来源: triz-40-principles, 2024-03-10]

## 教学场景高频引用的10个原理

| 编号 | 名称 | 含义 |
|---|---|---|
| 1 | 分割（Segmentation） | 把对象分成独立部分，或做成可拆卸结构 |
| 2 | 抽取（Extraction） | 抽出对象中"有害"或"必要"的部分 |
| 3 | 局部质量（Local Quality） | 让对象各部分承担不同功能，各自处于最有利状态 |
| 4 | 不对称（Asymmetry） | 把对称对象变为不对称，或增加不对称程度 |
| 5 | 组合（Combining） | 在空间/时间上把相同或相关的对象组合起来 |
| 10 | 预先作用（Prior Action） | 预先对对象施加变化，或预先安排好所需动作 |
| 13 | 反向作用（Do it in Reverse） | 用相反的动作代替原动作 |
| 15 | 动态化（Dynamics） | 让对象的特性在工作过程中自动调整到最佳状态 |
| 35 | 参数变化（Parameter Changes） | 改变对象的物理状态、浓度、柔性等参数 |
| 40 | 复合材料（Composite Materials） | 把单一材料变为复合材料 |

[来源: triz-40-principles, 2024-03-10]

## 用法

40个原理不是孤立查表用的清单，而是与 [[triz-contradiction-matrix]] 配套：先把问题抽象为技术矛盾，查矩阵得到推荐原理编号，再把原理映射回具体工程方案。[来源: triz-40-principles, 2024-03-10]

## 边界

- 40个原理是启发式方向，不是自动生成方案的算法，仍需工程师结合领域知识落地。[来源: triz-40-principles, 2024-03-10]
- 原理编号和命名在不同译本中略有差异（如"局部质量"也译作"局部性质"），本知识库以教学通用译名为准。[来源: triz-40-principles, 2024-03-10]
