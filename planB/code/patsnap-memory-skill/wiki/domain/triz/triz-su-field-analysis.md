# TRIZ 物场分析

**物场分析**（Substance-Field Analysis，简称 Su-Field）是 TRIZ 中用来建模"问题结构"的图形化工具，把技术系统的最小功能单元抽象为三个元素：S1（作用物质/对象）、S2（工具物质）、F（场，连接S1和S2的作用方式，如机械场、热场、电场、磁场、化学场）。[来源: triz-su-field-analysis, 2024-03-15]

## 三种基本模型

1. **完整有效模型**：S2 通过 F 对 S1 施加了期望作用，系统正常工作。
2. **有害作用模型**：S2 通过 F 对 S1 产生不期望的副作用，需消除或屏蔽。
3. **不足/缺失模型**：S1 需要某种作用但 F 或 S2 缺失/不足，需补全。

[来源: triz-su-field-analysis, 2024-03-15]

## 用法

物场分析常与76个标准解（Standard Solutions）配套使用：把问题画成物场模型后，对照标准解模板快速定位改造方向。这是与 [[triz-40-principles]] 平行的另一条 TRIZ 工具线——40原理面向"技术矛盾"，物场分析+标准解面向"功能结构缺陷"。[来源: triz-su-field-analysis, 2024-03-15]

## 边界

- 物场分析假设问题能被抽象为"物质-场"三元结构，对纯信息类、组织流程类问题的适用性较弱，通常需先转化为物理/工程视角。[来源: triz-su-field-analysis, 2024-03-15]
