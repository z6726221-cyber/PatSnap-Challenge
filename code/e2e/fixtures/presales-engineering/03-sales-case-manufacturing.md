# 制造业客户研发情报拜访案例

在制造业客户售前场景中，销售通常先确认三类问题：研发团队如何发现外部技术线索、专利检索结果如何转化为研发决策、跨区域团队如何复用已有分析。

推荐开场口径：

“我们不是替代研发判断，而是把专利、技术情报和内部知识变成可复核的分析过程，让团队更快知道哪些路线值得继续看、哪些证据需要回到原始来源确认。”

常见异议处理：

- 如果客户担心 AI 输出不可控，需要强调引用来源、人工复核和边界说明。
- 如果客户已有专利数据库，需要把对话式检索、报告生成和团队知识复用作为差异点。
- 如果客户关注数据安全，需要确认部署方式、权限边界和资料可见范围。
