# Analytics AI Mode 产品能力口径

Analytics AI Mode 面向企业 IP、研发和战略团队，提供自然语言专利检索、专利解读、竞争技术线索整理和可追溯报告生成能力。系统回答必须保留引用来源，用户可以回到专利、项目资料或企业知识条目复核。

核心价值包括：

- 把“关键词检索 + 人工筛选 + 手工报告”的流程压缩为可追溯的分析对话。
- 对专利摘要、权利要求、技术主题进行结构化解读，帮助非专利背景成员更快理解技术含义。
- 支持把分析结果整理为可分享报告，适用于研发立项、竞品技术路线跟踪和客户沟通。

能力边界：AI Mode 不能替代专利法律意见；对客户内部业务目标、采购流程和未公开研发计划只能形成假设，必须由销售或售前在拜访中验证。
