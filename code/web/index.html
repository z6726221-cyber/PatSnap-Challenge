一句话解释

TRIZ 可以理解为研发团队的系统化解题方法：先识别技术矛盾，再参考可复用的发明原理，最后形成需要验证的方案方向。

能力边界

TRIZ 不是自动给出唯一正确答案，也不能替代实验验证、工程评审或专利法律判断。
