# Analytics AI Mode 产品能力

Analytics AI Mode 支持用自然语言提出专利检索和技术分析问题，将检索结果、专利解读、技术主题和竞争情报整理成可复核的报告。核心能力包括专利检索、专利解读、竞争技术线索归纳、来源引用和报告输出。

适用场景包括企业 IP 团队日常检索、研发立项前的技术路线观察、销售或解决方案团队准备客户材料。系统强调来源可查，用户可以回到原始专利、项目资料或内部知识条目确认。
