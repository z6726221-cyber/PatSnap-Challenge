# Analytics AI Mode 七月运营素材

七月传播主题：让专利分析进入 AI Mode。

建议渠道：

- 官网活动页：适合放 30 秒产品预热视频和报名入口。
- 公众号：适合发布长图文和视频号同步。
- 销售转发：适合用短文案引导客户预约演示。

素材关键词：自然语言检索、来源可查、专利解读、竞争技术洞察、报告可复核。
