# FAQ：AI Mode 结果如何追溯

客户问：AI Mode 生成的结论可以追溯吗？

建议回答：可以。销售沟通时要强调“结论不是凭空给出”，而是回到专利、研发资料或企业知识条目的引用来源。对于资料没有覆盖的判断，必须标为待核实或下一步验证问题。

不要这样说：不要承诺 AI 自动完成法律判断、采购判断或客户内部流程判断。
