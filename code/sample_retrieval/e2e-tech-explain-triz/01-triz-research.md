# TRIZ 方法论研发知识

TRIZ 是一种系统化创新方法论，核心思想是从大量发明案例中总结可复用的问题解决模式。它帮助团队把“遇到技术矛盾”转化为“识别矛盾、参考通用原理、生成可验证方案”的过程。

常见解释框架：

- 技术矛盾：改进一个指标时，另一个指标可能变差。
- 发明原理：从历史发明经验中抽象出的解决思路。
- 理想解：用更少资源获得更高价值的目标状态。

边界：TRIZ 不是自动生成正确答案的工具，也不替代实验验证、工程评审或专利法律判断。
