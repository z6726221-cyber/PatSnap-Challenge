# TRIZ 销售培训口径

给销售和售前讲 TRIZ 时，可以把它类比成“研发团队的系统化解题地图”：不是直接告诉你唯一答案，而是帮助团队把模糊问题拆成矛盾、约束和可验证方向。

客户沟通价值：

- 适合研发创新、产品改良、工艺优化场景。
- 可以帮助客户把零散经验沉淀成可复用方法。
- 与专利和技术情报结合时，可以辅助发现可参考的技术路线。

需要提醒销售：类比只用于理解，不能把“解题地图”说成产品功能或技术事实。
